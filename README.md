## Getting Started

To get started with **Toon Tracker**, clone this repository and follow these commands:

```bash
# Install dependencies
pnpm install
```

```bash
# Run the development server
pnpm run dev
```
or

```bash
# Run the desktop app
pnpm run tauri
```

- React: For a dynamic and responsive user experience.
